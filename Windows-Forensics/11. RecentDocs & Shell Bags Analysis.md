RecentDocs is a valuable Windows forensic artifact that records recently accessed files and folders. It helps investigators understand user activity and reconstruct actions performed on a system.

## ğŸ“ What Is RecentDocs?

RecentDocs is a Windows Registry key that stores the names of files and folders recently opened by the logged-in user.

**Registry Path:** `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\RecentDocs`

This key contains lists of recently accessed items organized by file extension (e.g., `.docx`, `.pdf`, `.jpg`) and a general list for folders or mixed items.

---

## ğŸ¯ Why RecentDocs Matters in Forensics

**RecentDocs can help investigators:**
* Identify recently opened files and documents
* Validate user activity, such as access to sensitive data
* Support cases involving intellectual property theft or data misuse
* Reconstruct timelines of document interaction

It is especially useful when examining what a user interacted with shortly before an incident.

While powerful, RecentDocs has some constraints:

**ğŸ”¹ Limited Storage**

Shellbags are a powerful Windows forensic artifact used to understand which folders a user accessed, how they viewed them, and even whether the folders still exist. They can reveal user activity long after files or directories are deleted, making them extremely valuable during investigations.

## ğŸ“Œ What Are Shellbags?

Shellbags are Windows Registry entries that store details about folders viewed in File Explorer.

**They record:**
* Folders accessed by the user
* Folder path structures (including deleted ones)
* View settings (icons, list, details, etc.)
* Window size and position
* Access to local, removable, and network directories

This makes Shellbags very useful for reconstructing user behavior, verifying hidden activity, and uncovering evidence of deleted folder access.

## ğŸ§© Why Shellbags Matter in Forensics

**Shellbags help investigators:**
* Identify folders opened by a user, even if the folder no longer exists
* Understand directory traversal patterns
* Recover previous paths of renamed, moved, or deleted folders
* Gain visibility into external drives or network shares accessed

They are often used in cases involving data hiding, unauthorized access, or evidence tampering.
