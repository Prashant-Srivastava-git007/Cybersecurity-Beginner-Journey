Volatility is an open-source memory forensics framework widely used by DFIR, SOC, and malware analysts. It is built in Python and uses a collection of plugins to analyze memory dumps efficiently.

# ğŸ“ 1. System Information

### **Plugin: windows.info**

Displays basic OS information such as version, architecture, build, and install date.

```
python vol.py -f path/to/memory_dump.raw windows.info > path/to/output/systeminfo.txt
```

---

# ğŸ§µ 2. Process Analysis

## **a) Process Tree**

### Plugin: windows.pstree

Shows processes in parent-child hierarchy.

```
python vol.py -f path/to/memory_dump.raw windows.pstree > path/to/output/pstree.txt
```

---

## **b) Process List**

### Plugin: windows.pslist

Lists running processes with essential attributes (PID, PPID, name, creation time).

```
python vol.py -f path/to/memory_dump.raw windows.pslist > path/to/output/pslist.txt
```

Filter by a specific PID:

```
python vol.py -f path/to/memory_dump.raw windows.pslist --pid <PID> > path/to/output/pslist_pid.txt
```

---

## **c) Dumping a Process**

```
python vol.py -f path/to/memory_dump.raw windows.pslist --pid <PID> --dump
```

Extracted processes can be used for malware analysis.
