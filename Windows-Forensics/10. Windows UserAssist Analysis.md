## ðŸ“Œ What Are UserAssist Entries?

UserAssist entries are Windows Registry records that track the execution of GUI-based applications.

**They help forensic analysts understand:**
* Which applications were opened
* How frequently they were used
* When they were last executed
* User behavior patterns and activity timelines

These entries are useful in cases of insider threats, malware execution, and activity reconstruction.
