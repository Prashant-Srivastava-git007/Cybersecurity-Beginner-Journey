# üîç Mozilla Firefox Forensic Artifacts Guide

Mozilla Firefox stores most user browsing artifacts inside the following directory:

```
C:\Users\<Username>\AppData\Roaming\Mozilla\Firefox\Profiles\<ProfileFolder>\
```

This folder contains multiple SQLite databases, JSON files, and session data useful for browser forensics.

---

## üìÅ Key Firefox Artifact Files

| File / Directory                  | Description                                          | Type          |
| --------------------------------- | ---------------------------------------------------- | ------------- |
| **places.sqlite**                 | Browsing history, visit timestamps, and bookmarks.   | SQLite        |
| **logins.json / key4.db**         | Saved passwords (encrypted) and credential metadata. | JSON / SQLite |
| **cookies.sqlite**                | Website cookies stored by Firefox.                   | SQLite        |
| **extensions.json / extensions/** | Installed extensions and their metadata.             | JSON / Folder |
| **favicons.sqlite**               | Website favicon data linked to visited URLs.         | SQLite        |
| **sessionstore-backups/**         | Session/tab recovery data (JSONLZ4 format).          | Folder        |
| **formhistory.sqlite**            | Saved form and search bar entries.                   | SQLite        |

---

## üõ† Viewing Firefox SQLite Artifacts

Most Firefox databases (e.g., `places.sqlite`, `cookies.sqlite`, `favicons.sqlite`) can be opened using:

* **DB Browser for SQLite**
* **SQLiteStudio**
* Any forensic browser artifact parser

These tools allow you to view tables such as `moz_places`, `moz_historyvisits`, `moz_cookies`, etc.

# üîé BrowsingHistoryView (NirSoft)

**BrowsingHistoryView** is a lightweight tool that extracts browsing history from multiple browsers, including:

* Firefox
* Chrome
* Microsoft Edge
* Internet Explorer
* Safari (if installed)

### ‚úÖ Key Features

* Extracts history records (URLs, visit time, visit count, referrer, etc.)

* Exports results to **HTML, CSV, XML, or TXT**

* Combines history from multiple browsers into one view

## ‚≠ê Additional Notes 

* Firefox may create **multiple profiles**; choose the one with the most recent timestamps.
* Some files (like saved passwords) are **encrypted** and require Windows account context to decrypt.
* Jsonlz4 session files can be decompressed using tools like **Firefox Decrypt** or **Mozilla‚Äôs lz4 libraries**.
* Always make a **copy** of the profile folder before analysis to avoid modifying timestamps.
